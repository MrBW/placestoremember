Spezifikation "Places to remember"

Ziel der Anwendung soll es sein, eine Liste von Orten mit einem Timestamp zu speichern, an dem der Ort zum letzten Mal vom Benutzer besucht wurde.

GIVEN

Über eine REST API werden die folgenden Werte in einer Datenbank persistiert.

Name Last visit
Codecentric Solingen 19/08/2014
Codecentric Münster 24/06/2015
Codecentric Frankfurt
Codecentric München 18/08/2014
Codecentric Serbien 23/05/2015

WHEN

Wenn ich die folgende URL http://localhost:8080 der Anwendung aufrufe

THEN

Sehe ich die folgenden 5 Elemente mit ihren entsprechenden Werten.

Name Last visit
Codecentric Solingen 1 year, 3 month and 3 days ago
Codecentric Münster 28 days ago
Codecentric Frankfurt you´ve never been there
Codecentric München 11 month and 4 days ago
Codecentric Serbien 11 month and 13 days ago
